# Student Performance Prediction System

A Machine Learning web application that predicts whether a student will PASS or FAIL based on academic factors.

## Features

• Machine learning model using Scikit-learn
• Flask web application
• Interactive UI for predictions
• Data visualization using Matplotlib

## Tech Stack

Python
Pandas
NumPy
Scikit-learn
Flask
Matplotlib

## Project Workflow

1. Data preprocessing
2. Exploratory Data Analysis
3. Model training
4. Model evaluation
5. Model deployment

## Model Used

Logistic Regression

Accuracy: ~85%

## Run Locally

Install dependencies

```bash
pip install -r requirements.txt
```

Train model

```bash
python src/train_model.py
```

Run web app

```bash
python app.py
```

Open browser

```
http://127.0.0.1:5000
```
